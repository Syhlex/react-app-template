# react-app-template

Starter template for React supporting TypeScript, modular Sass, and Jest. Bundled with webpack.

## Getting started

Install dependencies.

```bash
yarn install
```

Setup Git hooks.

```bash
yarn setup-git-hooks
```

## Scripts

Start the development server.

```bash
yarn start
```

Generate a production build.

```bash
yarn build
```

Run unit tests.

```bash
yarn test
```
